# Node-daemon archive

These documents record audits, specifications, acceptance notes, and release
planning from the superseded TypeScript/npm product. They are historical
evidence, not current build, release, deployment, or product instructions.

The implementation under `prettyd/` remains temporarily valuable as a protocol
compatibility and rollback reference while the production mini still runs the
Node daemon. Current work starts with the repository root `AGENTS.md`,
`STATE.md`, `docs/WHY.md`, and `docs/NATIVE_APP.md`.
